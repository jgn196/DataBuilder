Version 1.0.0.0 was released on 30/12/2014.

Changes in This Release

This is the first release of the library.

The DataBuilder class supports building data using:

  • Raw data supplied as arrays of bytes.

  • 8, 16, 32 and 64-bit integers, both signed and unsigned.

  • Integers encoded using either little or big endian byte orders.

  • Text strings encoded as ASCII, UTF-7, UTF-8, UTF-16 (both little and big endian) and UTF-32.

  • Repeating patterns using nested DataBuilder objects.
