Remarks

Clients instantiate an instance of the builder, call various methods in the Fluent style to configure how data will be generated and then call the Build method to build the data.

The order in which methods are called is the order in which their arguments will be added to the finished built data.

Examples

This example creates a simple binary header for a hyperthetical file format.

The header has 3 marker bytes that spell "FOO" in ASCII, followed by 2 16-bit unsigned integers that contain the major and minor file format version. Finally, the header contains an unsigned 64-bit integer that holds the size of data in the file.

Copy
byte[] header = new DataBuilder()
  .AppendText("FOO") // Marker for the FOO format
  .AppendUInt16(1)   // Major version 1
  .AppendUInt16(0)   // Minor version 0
  .AppendUInt64(data.Length)
  .Build();
